\doxysubsection{Detailed Description}
Header file for SHA-\/3 and SHAKE cryptographic hash functions (FIPS 202 standard). 

This file provides the interface for SHA-\/3 based functions including SHAKE128, SHAKE256, SHA3-\/256, and SHA3-\/512, along with necessary constants, types, and function prototypes.

It defines the \doxylink{structkeccak__state}{keccak\+\_\+state} structure used for internal state management, and the FIPS202\+\_\+\+NAMESPACE macro for namespace management of function names.

The implementation follows the FIPS 202 standard for cryptographic hash functions and extendable-\/output functions (XOFs).

The actual function implementations are located in shared libraries (.so files), primarily in\+:
\begin{DoxyItemize}
\item {\ttfamily libpqcrystals\+\_\+fips202\+\_\+ref.\+so} (reference implementation)
\item {\ttfamily libpqcrystals\+\_\+fips202x4\+\_\+avx2.\+so} (AVX2-\/optimized implementation)
\end{DoxyItemize}

\begin{DoxyNote}{Note}
All functions operate on byte arrays, and sizes are specified in bytes.
\end{DoxyNote}

\end{DoxySeeAlso}

Computes the SHAKE128 extendable-\/output function (XOF) on input data. 

This function takes an input message {\ttfamily in} of length {\ttfamily inlen} bytes and produces an output of {\ttfamily outlen} bytes into the buffer {\ttfamily out}.

SHAKE128 is a member of the SHA-\/3 family of hash functions standardized by NIST, providing cryptographic hashing with arbitrary output length.
