\doxysubsection{Detailed Description}
This file defines the {\ttfamily poly} data structure and provides function declarations for operations on polynomials used in cryptographic algorithms, particularly in the context of Kyber, a lattice-\/based encryption scheme. 

The operations include compression, decomposition, NTT (Number Theoretic Transform), base multiplication, and polynomial reductions.

The {\ttfamily poly} structure represents a polynomial used in the Kyber cryptosystem, and the functions in this file perform various polynomial operations, such as converting to/from byte arrays, applying noise, performing modular reductions, and executing arithmetic operations (addition, subtraction, multiplication).

The function implementations are likely found in the corresponding Kyber implementation files.
