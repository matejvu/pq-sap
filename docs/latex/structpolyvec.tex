\doxysubsection{Detailed Description}
A structure that holds a vector of polynomials. 

The {\ttfamily polyvec} structure contains an array of {\ttfamily KYBER\+\_\+K} polynomials, where {\ttfamily KYBER\+\_\+K} is a predefined constant indicating the number of polynomials in the vector.
